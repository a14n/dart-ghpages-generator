Generator class

This class allows to generate a new version of gh-pages. You can choose what you want to put in gh-pages : examples, dartdoc, docs and/or custom files.

As sample here's how the gh-pages for this package is generated :

new gh.Generator()
..setDartDoc(['lib/ghpages_generator.dart'], excludedLibs: ['path'])
..templateDir = 'gh-pages-template'
..generate();
